Download

The current version of the UMC IDE is 2.0.1

The UMC IDE is available in two flavours: eclipse plugin and standalone application.

  • * Eclipse plugin: Use the following link as the update site (Copy-paste it into Eclipse's update window): http://eclipse.wesnoth.org/updates

  • * Standalone application:
    Download for Windows: 32 bit 64 bit | (Download post-update fix)
    Download for Linux: 32 bit 64 bit | (Download post-update fix)
    Download for Mac OS X: 64 bit | (Download post-update fix)

    * Note: The IDE does not support 32 bit Mac OSX , because it requires Java SE6 which is available just on the 64 bit Mac OS X.
    * Note: If the IDE doesn't start anymore after the update, please download and run the Post-Update fix script provided for your operating system.

    The application requires the Java runtime environment, so you need to install SUN's Java first (not openJDK or others) - Download Java.
